# [Q136]只出现一次的数字
---
给定一个**非空**整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

**说明：**

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

**示例 1：**
+ **输入：**[2,2,1]
+ **输出：** 1

**示例 2：**
+ **输入：**[4,1,2,1,2]
+ **输出：** 4
### 思路
a xor a = 0
a xor b xor a = b

题目中说明无关数字均出现2次，即将数组遍历并异或后结果为目标数字


---
### Java代码实现
```java
class Solution {
    public int singleNumber(int[] nums) {
        int single = 0;
        for(int num : nums){
            //single = single ^ num;
            single ^= num;		//异或运算
        }
        return single;
    }
}
```
### 复杂度分析

+ 时间复杂度：O(n)。
+ 空间复杂度：O(1)。
